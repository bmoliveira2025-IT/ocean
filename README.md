# ocean.io 🌊🐍

A high-performance, visually rich browser game inspired by Slither.io, built with React, Vite, and HTML5 Canvas.

## Features
- **Fluid Gameplay**: Optimized for a locked 60 FPS even with massive snakes.
- **Dynamic Camera**: Smooth zooming that adapts to your growth.
- **Growth Speed Bonus**: Real speed increases as you grow to keep the game fast-paced.
- **Neon Aesthetic**: Beautiful underwater theme with glowing SVG graphics and particles.
- **AI Bots**: Challenging bots with diverse behaviors (hunting, foraging, fleeing).
- **Procedural Audio**: Dynamic sound effects and ambient music generated via Web Audio API.
- **Mobile Support**: Virtual joystick and boost button for touch devices.

## Tech Stack
- **React**: UI management and state.
- **Vite**: Ultra-fast build tool and dev server.
- **Tailwind CSS**: Modern styling.
- **PostCSS**: Processed styles for maximum compatibility.

## Getting Started
1. Clone the repository.
2. Run `npm install`.
3. Start the dev server with `npm run dev`.
4. Open `http://localhost:5173`.

Enjoy the ocean!
